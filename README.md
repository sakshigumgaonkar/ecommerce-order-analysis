# E-commerce Order Analysis

## Overview
This repository contains a multi-phase analysis of an e-commerce marketplace dataset
(OLIST).

The project begins with business-aware exploratory data analysis (EDA) focused on
data quality, order lifecycle validation, and baseline customer behavior.

## Project Scope & Roadmap

### Phase 1: Exploratory Data Analysis (Current)
- Dataset overview and integrity checks
- Business-aware missing value analysis
- Order lifecycle validation
- Product data completeness assessment
- Baseline customer repeat behavior analysis

### Phase 2: Customer Behavior Analysis (Planned)
- Repeat purchase drivers
- Customer segmentation
- High-value customer analysis

### Phase 3: Fulfillment & Operations (Planned)
- Delivery delays and SLA compliance
- Seller-level performance patterns

### Phase 4: Product & Conversion Insights (Planned)
- Product description impact on reviews and returns
- Category-level conversion signals
- Pricing and freight sensitivity

> Note: This repository currently contains Phase 1 only.
> Subsequent phases will be added as separate notebooks.
